> 2026-05-10 작성 / 사용자 김진호 / 모바일 앱 (Flutter 예정)

## 1. 앱 핵심 가치

Claude가 시장 정보(시세·공시·뉴스·유튜브)를 자동으로 모아서, 관심종목은 깊이 분석하고 그 외 종목은 이벤트성 뉴스로 빠르게 캐치해주는 앱.

**가장 중요한 차별점:** 주가에 영향 주는 이벤트성 뉴스를 빠르게 잡아내기.

---

## 2. 화면 구성 (탭 3개)

### [탭 1] 관심종목
- 그룹별로 종목 정리 (관심1, 관심2, 관심3 …)
- **그룹은 사용자가 자유롭게 추가/삭제/이름변경 가능**
- 초기 그룹: VIP 관심 / 지수우승 / vvip(추정) / 국내 ETF / 우주항공 (총 5개, 약 50종목)
- 종목 누르면 → **기업 분석 페이지로 바로 이동**
- 매일 자동 분석 업데이트

### [탭 2] 주식뉴스
- 시장 영향력 있는 이벤트 뉴스 피드
- 각 뉴스에 "관련 종목" 자동 추출 (Claude)
- 종목별 액션 버튼 3개:
  - **[관심종목 추가]** → 관심 그룹 선택 → 풀 분석 시작
  - **[보관]** → 보관 탭으로 (가벼운 추적)
  - **[삭제]** → 무시
- 새로고침은 5~10분 단위 자동

### [탭 3] 보관
- "보관" 처리한 종목들
- 기업 분석 리포트 열람 가능
- 관심종목보다 가벼운 추적 (분석 빈도 ↓)
- 마음에 들면 → 관심종목으로 승격 가능

## 3. 종목 상세 페이지 (관심종목 누를 때)

종목 상세 화면 구성:

### 3-1. 헤더
- 종목명 / 종목코드 / 시장 / 업종
- 현재가, 등락률, 거래량
- 최근 6개월 차트

### 3-2. 기업 분석 (Claude 자동 생성, 4개 섹션)
1. **사업 내용 + 매출 구조** — 뭐 하는 회사, 사업부문별 매출 비중
2. **재무 지표** — PER / PBR / ROE / 부채비율 (동종업계 평균 대비)
3. **최근 실적 추이** — 4~8분기 매출·영업이익 그래프
4. **경쟁사 비교 + 적정주가 추정** — 동종업계 비교, Claude 종합 의견

### 3-3. 관련 뉴스
- 이 종목 관련 최신 뉴스 5~10개
- 각 뉴스 핵심 1줄 요약

### 3-4. 내 메모 ⭐ (신규 추가)
- 사용자가 자유롭게 작성하는 메모란
- "왜 관심 갖게 됐는지", "매수 시점 전략", "체크포인트" 등 개인 노트
- 검색 가능, 작성/수정 시간 기록
- (선택) 메모 내용도 Claude 분석에 컨텍스트로 활용 가능

### 3-5. 액션 버튼
- 관심 그룹 이동
- 보관으로 이동
- 관심종목에서 제거

## 4. ETF 처리 방침

ETF는 일반 종목과 분석 방식이 다름. **ETF 전용 리포트 형식 적용:**
- 추적 지수 / 운용보수 / 거래량
- 상위 보유 종목 TOP 10
- 동종 ETF 대비 수익률 비교
- (사용자 결정 대기 중 — 옵션 A/B/C)

## 5. 데이터 소스 (백엔드)

| 종류 | 소스 | 무료 |
|-----|------|-----|
| 한국 시세 | pykrx / 네이버 금융 | O (15~20분 지연) |
| 공시 | DART OpenAPI | O |
| 뉴스 | 네이버 검색 API / 빅카인즈 | O (한도 있음) |
| 유튜브 | YouTube Data API + 자막 | O |
| 분석 | Claude API (Haiku 위주) | 유료 |

## 6. 기술 스택 (예정)

- **백엔드:** Python + FastAPI
- **DB:** Supabase (PostgreSQL)
- **호스팅:** Railway / Fly.io 무료 티어
- **푸시:** Firebase Cloud Messaging
- **앱:** Flutter (iOS/Android 동시)
- **스케줄러:** APScheduler
