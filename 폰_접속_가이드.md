# 📱 폰에서 주식정보 앱 사용하기

## 필요한 것
- ✅ PC 와 폰이 **같은 와이파이** 에 연결되어 있을 것
- ✅ PC 에서 `2_웹앱_띄우기.bat` 실행 중일 것

---

## 1단계: PC IP 주소 확인

`2_웹앱_띄우기.bat` 더블클릭하면 cmd 창에 이런 줄이 나와요:

```
IPv4 주소 . . . . . . . . . . . . : 192.168.1.10
```

이 숫자(`192.168.1.10`)가 PC IP 주소예요. **마지막 숫자는 본인 PC마다 다릅니다.**

---

## 2단계: 폰 브라우저에 입력

폰의 Chrome / Safari / Edge 등에서 주소창에:

```
http://192.168.1.10:8501
```

(본인 PC IP 주소 + `:8501`)

→ Enter → 웹앱 화면 뜸

### 홈 화면 바로가기 만들기 (앱처럼)

**iPhone (Safari)**
1. 페이지 열린 상태에서 하단 공유 버튼 (📤)
2. "홈 화면에 추가"
3. 이름 입력 ("주식정보") → 추가
4. 홈 화면에 아이콘 생김 → 누르면 앱처럼 풀스크린

**Android (Chrome)**
1. 페이지 열린 상태에서 우상단 메뉴 (⋮)
2. "홈 화면에 추가"
3. 확인

이러면 폰에 진짜 모바일 앱 같은 아이콘이 생겨요.

---

## 3단계: 처음 실행 시 — Windows 방화벽 허용

PC에서 `2_웹앱_띄우기.bat` 처음 실행하면 알림 창이 뜰 수 있어요:

> "Windows Defender 방화벽이 일부 기능을 차단했습니다."

**"액세스 허용"** 클릭. (안 하면 폰에서 접속 불가)

---

## 폰에서 접속 안 될 때 체크리스트

1. **PC와 폰이 같은 와이파이?** — 확인 필수
2. **IP 주소 정확?** — `ipconfig` 다시 확인
3. **방화벽 허용?** — Windows 보안 → 방화벽 → 인바운드 규칙에서 8501 포트 허용
4. **PC 절전 모드?** — 절전 들어가면 끊김. 절전 해제 OR 설정에서 절전 미루기
5. **VPN 켜져 있음?** — 끄거나 분할 터널링
